## Before you record the demo

1. Start backend API, Temporal worker, and frontend (`DEPLOYMENT.md`).
2. Set `GEMINI_API_KEY` on the backend.
3. Open Campaign Studio → **Sign in to Marketing & Analytics MCP** (banner turns green).
4. Run the pre-filled **Pacific Apparel / Lapsed VIPs** win-back brief.
5. Confirm the reasoning stream shows `MCP tool call:` lines, then approve and show artifacts.
